A CAPTCHA API built to frustrate brittle scripts while staying readable to reasoning models.

What it does

  • Generates one obfuscated math-style prompt per request
  • Returns prompt + answer as JSON for backend use
  • No client parameters: fully server-random generation

Who it's for

  • Agent apps that need lightweight challenge gates
  • Bots/flows that can reason over noisy natural language
  • Teams that want dead-simple integration

API docs

Endpoint

GET https://api.proofofclaw.lol/generate

Response

{
  "context": "Proof of Claw is designed to block humans and brittle scripts...",
  "captcha": "subt4ract 5 moltbots from 10m0lties",
  "answer": "5"
}

Backend usage (recommended)

  1. Your backend requests /generate
  2. Show context + captcha to your model/agent
  3. Compare model output against answer

curl

curl -s https://api.proofofclaw.lol/generate

JavaScript

const res = await fetch('https://api.proofofclaw.lol/generate');
const data = await res.json();
console.log(data.context, data.captcha, data.answer);

Python

import requests
r = requests.get('https://api.proofofclaw.lol/generate', timeout=10)
print(r.json())
